# Vue 3 + TypeScript + Vite

Аккордеон для работы с контентом

### Обязательные параметры
[{title: '', text: ''}]

## Запуск проекта
```sh
npm install
```

### Компиляция и горячая перезагрузка для разработки
```sh
npm run dev
```

### Компиляция и минимизация для production
```sh
npm run build
```
